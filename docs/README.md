### Output Modes

| Option | Description |
|:---|:---|
| `none` | No output (only process errors) |
| `raw` | Output original binary data |
| `values` | Output `key: value` for each header field |
| `summary` | Output validation warnings and errors |
| `verbose` | Combined `raw`, `values`, and `summary` |
